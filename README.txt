How to run the webapp:
1.- Unzip the folder
2.- Put the folder into the folder where you have the server running and open open it in your browser
3.- If you don't have a server, download a localhost server(we've user XAMPP)
3.1.- Download XAMPP https://www.apachefriends.org/es/index.html
3.2.- Install it
3.3.- Put the folders of the webapp into the dashboard folder of XAMPP
3.4.- Open your browser and write http://localhost/dashboard/remindme/ (The path may change depending on where you put it)
4.- Enjoy our webapp!

Hint! You need to login with Google to use our application
